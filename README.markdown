# re-Chronic #

## Introduction ##
A natural language date parser for .Net written in C#. Originally written by Mike Schrag as a direct port of Ruby's chronic.

re-Chronic is add support .Net Framework 4.6 and .Net Standard 2.0 to nChronic.
